# Kohan(湖畔)
>日本語特有の後置修飾を可能にするプログラム言語

# 動作の設計

サーバー格納データの形式
- .kld
１対１でデータを格納し、改行で区分け。
data_A>01
のような記述式
- .kkd
漢字データ格納に特化。読むときは各文字のUnicodeを用いて識別する。

>※設計意図
>漢字とそれ以外の違いを明確にすることで漢字の役割をわかりやすくするため。

- .kファイルを検知 
   1. 変更を検知
      1. コードデータをUnicodeから独自データに変換
         - コンパイラであるなら
            1. 既存言語に書き換えたデータを作成
            2. 依存関係を解決
            3. フォルダ構成のコピー
            4. 適切な形式で保存
